📋 데이터셋 테이블 편집기

📖 사용법 — 표에서 직접 행/컬럼을 추가·수정·삭제하고 [저장]하면 즉시 반영됩니다
  • 셀 수정: 셀 클릭 → 입력 → Enter 확정 / Esc 취소. 숫자·true/false·null·JSON(배열/객체)은 자동으로 타입이 보존됩니다 (예: ["별칭1","별칭2"]는 배열로 저장).
  • 행 추가: 상단 [+ 행 추가] — 현재 컬럼 구조의 빈 행이 맨 위에 생깁니다. 행 번호 옆 ⧉는 해당 행 복제, ✕는 행 삭제.
  • 컬럼 추가/수정/삭제: 상단 [+ 컬럼 추가]로 새 필드 추가. 컬럼 머리글에 마우스를 올리면 ✎(이름 변경 — 모든 행의 키가 함께 변경) / ✕(컬럼 삭제 — 모든 행에서 해당 키 제거)가 나타납니다.
  • 빈 셀: 값이 없는 셀은 저장 시 해당 키를 아예 넣지 않습니다(공백 문자열이 필요하면 "" 입력).
  • 저장: [💾 저장]을 누르면 표 전체가 파일의 새 내용이 됩니다. 서버가 백업 → 컴파일 검증 → 핫스왑을 수행하고, 문제가 있으면 자동 롤백 후 사유를 보여줍니다. 되돌리기는 JSON 관리 콘솔의 백업/복원을 사용하세요.
  • 검색: 검색어 입력 시 일치하는 행만 표시됩니다. 편집·저장은 전체 데이터 기준으로 동작합니다.
  • 중첩 구조가 깊은 overrides와 시스템 관리(feedback/audit) 데이터셋은 이 화면에서 제외됩니다 — JSON 관리 콘솔을 사용하세요.
⚠ 저장되지 않은 변경이 있습니다 — . [💾 저장]을 눌러 반영하거나 [다시 불러오기]로 취소하세요.
